# Выпускная квалификационная работа по программе цифровой кафедры РТУ МИРЭА

## Постановка задачи
Обучить нейронную сеть для классификации изображений, содержащих спилы бревен, по трем категориям сортности:
- 1 и 2 сорт
- 3 сорт
- дрова
